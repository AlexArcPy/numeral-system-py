# numeral-system-py
Convert from one numeric system to other in python.
Supports following:
* roman numerals

# Example of usage
```python
>>> import numeral_system
>>> numeral_system.roman.encode(7)
'VII'
>>> numeral_system.roman.decode('XXII')
22
>>>
```
